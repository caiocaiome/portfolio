Ferramenta de Denúncia Online

Desenvolvi uma ferramenta de denúncia online que tem como objetivo fornecer um meio seguro e anônimo para pessoas relatarem incidentes de forma confidencial. Esse projeto foi criado com o intuito de facilitar a comunicação de denúncias, especialmente em ambientes escolares e empresariais, proporcionando um espaço confiável e acessível.

A ferramenta oferece funcionalidades como formulários interativos e um sistema de armazenamento seguro para manter as informações privadas. Ela foi desenvolvida utilizando ferramentas web como HTML, CSS e JavaScript, com integração a back-end para processar os dados de maneira eficiente.

Este projeto destaca a importância da segurança da informação e da usabilidade em plataformas voltadas para o público, tornando o processo de denúncia mais acessível e confiável.
